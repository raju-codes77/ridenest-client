RideNest 🚗

Live Site: https://ridenest-server-orpin.vercel.app/

RideNest is a modern car rental and booking platform where users can explore available cars, view detailed information, and book vehicles easily. The platform includes secure authentication, protected routes, and a clean responsive UI built with modern web technologies.

🚀 Features
🔐 User authentication with Email/Password and Google Login using Better Auth
🚘 Browse available cars with detailed car information and booking options
📅 Users can book cars and manage their bookings from the My Bookings page
➕ Logged-in users can add their own cars through the Add Car private route
🔒 Protected private routes including:
Add Car
My Bookings
My Added Cars
🌐 Responsive and modern UI built with Tailwind CSS, DaisyUI, and Hero UI
🔔 Beautiful toast notifications using React Toastify
🗄️ MongoDB database integration for storing users, cars, and booking data
⚡ Fast and optimized application powered by Next.js
🛠️ Technologies Used
Next.js
Tailwind CSS
DaisyUI
HeroUI
MongoDB
React Toastify
Better Auth
